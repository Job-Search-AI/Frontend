# AI Career Finder – Frontend

AI 기반 채용 공고 추천 서비스의 프론트엔드입니다.  
사용자가 자연어로 희망 직무·지역·경력 등을 입력하면, 백엔드에서 추천한 결과를 채팅 형식으로 보여줍니다.

<br>

## 기술 스택

- Next.js (App Router)
- React
- CSS + 커스텀 UI 컴포넌트

<br>

## 주요 기능

- 채팅형 검색 인터페이스
- 사용자 입력에 따른 백엔드 API 연동
- 응답 대기 중 로딩 상태 표시
- 대화 히스토리 유지 및 후속 질문 흐름 처리

<br>

## 로컬 실행

```bash
npm install
npm run dev
```
